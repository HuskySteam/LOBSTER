## ğŸ§  What is LOBSTER?

Most AI coding tools are a **single agent in a box**. You type a prompt, it generates code, done. No review. No tests. No memory.

LOBSTER gives you a **full AI development team** â€” not one agent, but a coordinated system of specialized agents that **plan, build, review, test, and learn from each other**.

```

LOBSTER:
  âœ“ Architect designs the interface          [2s]
  âœ“ Coder implements JWT auth service        [8s]
  âœ“ Coder adds rate limiting middleware      [5s]
  âœ“ Reviewer catches missing input validation [3s]
  âœ“ Coder fixes validation                  [4s]
  âœ“ Tester writes & runs 12 tests           [6s]
  âœ“ All agents pass â€” shipping it
```

It's also a complete agentic coding CLI: **40+ tools**, multi-provider LLM support, persistent memory, cost tracking, and automatic context injection.

Built as a fork of [OpenCode](https://github.com/anomalyco/opencode) with full [Claude Code](https://github.com/anthropics/claude-code) plugin compatibility.

| **5. Review Loop** | Coder â†’ Reviewer â†’ Tester â†’ fix â†’ repeat until all pass |
